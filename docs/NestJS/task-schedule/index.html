Task Schedule

Install package#

npm install --save @nestjs/schedulenpm install --save-dev @types/cron

Import package#

在app.module加[ScheduleModule.forRoot(), TasksModule]在tasks.module加providers: [TasksService]在tasks.service加tasks

Parmeter#

| | | | | |
| | | | | day of week
| | | | month
| | | day of month
| | hour
| minute
second (optional)

            • 每一秒
              45 * * * * 每分鐘，在第 45 秒
              0 10               * * 每小時，在第 10 分鐘開始時
              0               /30 9-17 * * 上午 9 點至下午 5 點之間每 30 分鐘一班
              0 30 11               * 1-5 週一至週五上午 11:30

45 1 16 23 10 (五個數值)

45:在每個 45 秒
1: 在每個 1 分
16: 在每個 16 時
23: 在每個月的 23 號
10: 在每個 10 月

@Cron('45 * * * * *'): 排程 //範例:每個 45 秒
@Interval(10000): 重複 //範例:每 10 秒
@Timeout(5000): 運行一次 //範例:程式執行後的 5 秒執行一次就結束
