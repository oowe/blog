Reference from https://www.runoob.com/w3cnote/yaml-intro.html

  • YAML 全名: YAML Ain't a Markup Language (意思為另一種標記語言)

  • 附檔名為: .yml, .yaml

基本語法#

  • 大小寫為不同東西
  • 縮排表示層級關係
  • 縮排單位為空白鍵、不得使用 tab
  • 縮排格數沒有硬性規定，同一層級為相同縮排即可
  • #表示註解

數據類型#

  • 對象: 為 key-value 的集合或對映 (key-value)
  • 數組: 一組按照序列排序的值又稱為序列 [list]
  • 純量: 單個不可再拆分

YAML 對象#

表示方法 1:

  • key-value 以 key: value 表示（冒號後面要空一格）

表示方法 2:

  • 也可以表示成 key:{key1: value1, key2: value2, ...}。
key:  child-key: value  child-key2: value2

表示方法 3: 較為複雜的對象格式，可以使用問號加一個空白表示複雜的 key，配合一個冒號加一個空白表示一個 value:

? - complexkey1  - complexkey2: - complexvalue1  - complexvalue2
info

因 markdown 自動縮排問題，上述縮排有些問題需更正
? 是一行

  • complexkey1 是下一行
    : 是一行
  • complexvalue1 是下一行

意思為對象的屬性是一個數組 [complexkey1,complexkey2]，對映的值也是一個數組[complexvalue1,complexvalue2]

YAML 數組 []#

-代表一個數組

下列為一個 Example:

apiVersion: v1kind: Podmetadata:  name: my-pod  labels:    app: webserverspec:  containers:    - name: pod-demo      image: zxcvbnius/docker-demo      ports:        - containerPort: 3000

Example 轉成 Json

{  "apiVersion": "v1",  "kind": "Pod",  "metadata": {    "name": "my-pod",    "labels": {      "app": "webserver"    }  },  "spec": {    "containers": [      {        "name": "pod-demo",        "image": "zxcvbnius/docker-demo",        "ports": [          {            "containerPort": 3000          }        ]      }    ]  }}

純量#

包含字串、布林、整數、浮點數、Null、時間、日期

boolean:  - TRUE #true,True都可以  - FALSE #false，False都可以float:  - 3.14  - 6.8523015e+5 #可以使用科学计数法int:  - 123  - 0b1010_0111_0100_1010_1110 #二進制表示null:  nodeName: "node"  parent: ~ #使用~表示nullstring:  - 哈哈  - "Hello world" #單引號或雙引號都可以  - newline    newline2 #字串可以多行，換行會轉成一個空白date:  - 2018-02-17 #日期必须使用ISO 8601格式，即yyyy-mm-dddatetime:  - 2018-02-17T15:02:31+08:00 #时間使用ISO 8601格式，時間和日期之間使用T連接，最後使用+代表時區

引用#

& 錨點和 * 别名，可以用來引用:

defaults: &defaults  adapter: postgres  host: localhost
development:  database: myapp_development  <<: *defaults
test:  database: myapp_test  <<: *defaults

等同於:

defaults:  adapter: postgres  host: localhost
development:  database: myapp_development  adapter: postgres  host: localhost
test:  database: myapp_test  adapter: postgres  host: localhost

&用來建立錨點（defaults），<< 表示合併到當前數據，*用来引用錨點。

- &showell Steve- Clark- Brian- Oren- *showell

轉成 Javascript

["Steve", "Clark", "Brian", "Oren", "Steve"];
