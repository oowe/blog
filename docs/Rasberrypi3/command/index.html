停用樹莓派的 dhcpcd 服務，改用傳統的 networking#

停用 dhcpcd 服務#

sudo systemctl disable dhcpcdsudo systemctl stop dhcpcd

啟用 networking 服務#

sudo systemctl enable networkingsudo systemctl start networking
sudo vim /etc/network/interfacesauto eth0iface eth0 inet staticaddress 192.168.99.2netmask 255.255.255.0gateway 192.,168.99.1dns-nameservers 8.8.8.8 168.95.1.1

啟用 ssh 服務#

sudo systemctl enable sshsudo systemctl start ssh

設定固定 IP#

在 Kernel 4.9 之後需要這樣設定
sudo vim /etc/dhcpcd.conf
interface eth0static ip_address=192.168.2.100/24 (完全沒有空白)static routers=192.168.2.1static domain_name_servers=192.168.2.1備註：上述方法的前提是 dhcpcd 和 networking 要開機時就預設啟動，指令如下：systemctl enable dhcpcdsystemctl enable networkingsetup auto connect to special wifi SSIDsudo /etc/wpa_supplicant/wpa_supplicant.conf
network={ssid="{ssid}"psk="{password}"key_mgmt=WPA-PSK}
network={ssid="{{ssid}"key_mgmt=NONE}
allow-hotplug wlan0iface wlan0 inet staticaddress 192.168.99.5netmask 255.255.255.0gateway 192.168.99.1wpa-conf /etc/wpa_supplicant/wpa_supplicant.conf

兩張網卡 forward 設定#

sudo vim /etc/sysctl.conf
# net.ipv4.ip_forward=1
net.ipv4.ip_forward=1
