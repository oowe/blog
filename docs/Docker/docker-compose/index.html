Docker Compose

安裝Docker-composesudo curl -L "https://github.com/docker/compose/releases/download/1.29.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-composesudo chmod +x /usr/local/bin/docker-composesudo ln -s /usr/local/bin/docker-compose /usr/bin/docker-compose#arm64sudo apt install docker-compose #在18.04支援這樣安裝


查詢版本docker-compose -vdocker-compose version 1.29.2, build 5becea4c
建立檔案vim docker-compose.yml
執行docker-composedocker-compose up -d-d 是指在背景執行
只執行特定容器docker-compose up -d ems-db


查詢psdocker-compose ps
查詢logdocker-compose logs <container_name>
停止docker-compose stop
移除所有docker-compose rm -fs-f表示不需再次詢問是否刪除提示-s會自動暫停所有正在運行的container
docker-compose set network in hostnetwork_mode: "host"

要加voluumesvolumes:- /home/ardomus/ems-opt/db-vol:/var/lib/mysql
comment 方式#

problem: Mac M1:Error:no matching manifest for linux/arm64/v8 in the manifest list entries#

  • 在Mac M1上遇到
# 原本的yamldb:    image: mysql:5.7    environment:    ...
# 請改成，就可以解決問題db:    image: mysql:5.7    platform: linux/amd64    environment:    ...

指定啟動的yaml檔名#

docker-compose -f {filename} up -d
