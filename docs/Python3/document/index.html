底線的意義
它代表著無名或未知的感覺(但不代表不存在)。像下面這個用法，當程式複雜時就可以用底線來代替一次性變數：def hello():    for _ in range(0, 15):        print(f"{_}")
_xx 隱藏此物件，不輕易被使用__xx 具有唯一性，無法被修改
        print(f"{_}")
=================================================#abc.pydef hello():    for _ in range(0, 15):        print(f"{_}")def _hello():    print('a')
若是import abc的方式abc._hello 還是可以呼叫但是如果from abc import *_hello()會失敗，不可以呼叫若要呼叫的話，需要from abc import _hello才可以---若是import class方式本來就可以拿到_hello

def foo2(value):    """纯return语句，相当于`return None`"""    if value:        return value    else:        return
def foo3(value):    """无return语句，也相当于`return None`"""    if value:        return value

tuple、list#

  • tuple
    • 是屬於不可變的資料型態(immutable)，當你的值都不會改變，而且你也不希望值被更動的時候，就是適合使用 tuple 的時候
    • 以圓括號代表，例如 a = (1,2,3)
    • 要特別注意若只有一個 tuple 元素要加,才會成立 (11,)，若(11)代表整數 11
    • 占用的空間比較少
  • list
    • 是可變，可使用 append()、remove()、pop()等會更動值的操作。
    • list 是 b = [1,2,3]

    • 放在 python_env/lib/python3.8/site-packages/_pytest/python_api.py

async await 說明#

  • async / await 是 Python 3.5+ 之後出現的語法，讓 Coroutine 與 Coroutine 之間的調度更加清楚
