  1. 註冊帳號後有一年免費的使用
    • EC2 (每個月 750 小時可以免費使用) 若你是兩個虛擬機就是兩倍小時在消耗。
  2. 登入後記得在右上角選擇東京，在開 EC2
  3. 選擇 Ubuntu 最新版/選擇綠色標籤的免費方案
    • instence 選擇 1 開一台
    • 硬碟免費是 8G （但之後都可以再加錢加裝，需要重開機就是了)
    • 設定 port，原則上全部 port 都是關閉，需要在 ecurity Group 設定)
    • key pair(.pem 檔案只能下載一次，一定要妥善保存)
    • 一開起來就會有 Public IP，但是是浮動的
    • 接下可以用 SSH 連線(要改 pem 的檔案權限為 400，用 pem 連不用打帳號密碼直接連線)
    • 虛擬機內部是使用虛擬 IP

MySQL 資料庫學習#

  • SQLlite 因為體積小，通常會放在嵌入式或是手機內
  • MariaDB 是 MySQL 作者在開發的，故完全相容可直接沿用，可以看 mysql 教程。
  • 資料庫可以想成 Excel 檔案，當然它的功能更強大
  • 寫程式就是在寫樣板，運行的時候再把資料餵進去
  • 用你寫的程式去管理 mysql 的程式，讓你不用自己刻
