TypeORM

Install TypeORM#

npm install --save @nestjs/typeorm typeorm mysql2

  • typescript 型別對應到資料庫型別

    • number -> integer
    • string -> varchar
    • boolean -> bool

  • 裝飾器

    • @Entity({name: 'school'}) 實體化 table
      • 默認以 name 參數爲 table name，若無則用 class name
      • 名稱命命名爲駝峰式會以下底線隔開（默認）
      • table name 都是小寫
    • @Column() 實體化屬性
    • @PrimaryColumn():主鍵
    • @PrimaryGeneratedColumn():更方便的主鍵
      • 假设你希望 id 列自动生成，你需要将@PrimaryColumn 装饰器更改为@PrimaryGeneratedColumn 装饰器

  • new table exmple

import { Entity, Column } from "typeorm";
@Entity({name: 'Photo'})export class Photo {  @PrimaryGeneratedColumn()  id: number;
  @Column({    length: 100  })  name: string;
  @Column("text")  description: string;
  @Column("double")  view: number;
  @CreateDateColumn({    type: 'datetime'  })  created: Date;
  @UpdateDateColumn({    type: 'datetime'  })  updated: Date;
