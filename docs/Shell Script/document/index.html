Reference from https://www.twblogs.net/a/5d088230bd9eee1ede03bd38

  • Shell script 副檔名為 .sh
  • 要賦予可執行的權限
    • chmod +x demo .sh (會將文件擁有者，文件所屬群組，對於其它用戶都會設定)
    • 可透過檔案詳細資料觀看有沒有+x 的執行權限
    • ls -l demo .sh
    • (example) -rw-rw-r-- 1 ardomus ardomus 1 七 14 18:01 demo .sh
    • (加權限)-rwxrwxr-x 1 ardomus ardomus 1 七 14 18:01 demo .sh
  • 權限腦補
    • r:讀取, w:寫入 x:執行
    • 第一個字符"-"指定了文件類型，在通常意義上，一個目錄也是一個文件，如果第一個字符是橫線，表示是一個非目錄的文件，如果是 d，表示是一個目錄。
      • (example)ls -l project
      • drwxrwxr-x 11 ardomus ardomus 4096 七 14 16:20 openleadr-python
    • 第二段 rwx 是『文件擁有者』的屬性
    • 第三段 rwx 是『文件所屬群組』的屬性
    • 第四段 rx 是『對於其它用戶』的屬性
    • 數字設定法
      • r ------------4
      • w -----------2
      • x ------------1
      • "-" ------------0
      • -rwxrwxrwx (777)每個人都有讀寫和執行的權限

Command#

  • cat /etc/shells //查看系統支援的 bash 種類及路徑
  • which bash
  • touch 可以建立檔案
  • #! /bin/bash
  • echo 'word'
  • sh hello .sh
  • ./hello .sh
  • 變數和註解
    • 就是 shell script 的註解方式
    • 變量分兩種: 系統制定和客戶制定
      • 系統都是全大寫 echo $BASH
        • echo $BASH_VERSION
        • echo $HOME
        • echo $PWD
      • 客戶通常都是小寫，但大寫也可以 (變數後面的等於不能空白)
        • name=Aaron
        • echo The name is $name
        • 變數不得為數字開頭
    • 讀取使用者輸入
      • echo "Enter name :"
      • read name1 name2 name3
      • echo "Entered name : $name1"
      • 可以寫成一行就好(簡化)
      • 可以讓輸入文字不要顯示(-s)
      • read -p 'username : ' user_var
      • read -sp 'password : ' pass_var
      • echo ""
      • echo "username : $user_var"
      • echo "password : $pass_var"
      • 可以寫成數組的方式
      • echo 'Enter name :'
      • read -a names
      • echo "Entered name : ${names[0]}, ${names[1]}"
      • 更簡潔的方式
      • echo 'Enter name :'
      • read
      • echo "Entered name : $REPLY"
    • 使用輸入參數($0 是程式檔名)
      • echo $1 $2 $3 ' > $1 $2 $3'
      • echo $0 $2 $3 ' > $1 $2 $3'
      • aaron chang kaa > $1 $2 $3
      • ./hello.sh chang kaa > $1 $2 $3
      • args=("$@")
      • echo ${args[0]} ${args[1]} ${args[2]}
      • echo $@
      • ./hello.sh aaron chang kaa
      • aaron chang kaa
      • echo $# (輸出參數數量)
    • IF 判斷式
      • format
      • if [ condition ] //[]中間的空格不能省略
      • then
      • statement[yes]
      • elif [ condition ]
      • then
      • statement [y]
      • else
      • statement[error]
      • fi
      • if [ $count == 10 ] //可以使用-qt 和==
      • if (( $count == 10)) //只能使用==
      • if [[$count == 10]]
      • if [[$count -eq 9]]
      • then
      • echo "bingo is 9"
      • elif [ $count -eq 10 ]
      • then
      • echo "bingo is 10"
      • else
      • echo "error"
      • fi
