Corrupted double-linked list 的問題#

warning

在切換不同 video 時，會有 corrupted double-linked list 的問題
主因：
是有重複的變數，在進入第二隻的影片時，會被重複宣告的問題
所以必須要先 release 後，才能被使用 

#frame的部份， cap必須要被releasecap.release()
#namewindow的部份，需要在while先被宣告，這樣就不會發生重複宣告cv2.namedWindow("%s_frame" %process_number)

讀取 rtsp#

cap = cv2.VideoCapture("rtsp://admin:admin@192.168.0.10:8554/CH001.sdp")while(cap.isOpened()):  ret, frame = cap.read()  if not ret:         break  else:         #cv2.imshow('frame', frame)         if cv2.waitKey(20) & 0xFF == ord('q'):         breakcap.release()cv2.destroyAllWindows()

problem: show image 不能超過 800*600#

請加上cap.set(cv2.CAP_PROP_FOURCC, cv2.VideoWriter_fourcc(*"MJPG"))
因為影像格式一進來是YUV， 最大只支援到800*600須改成MJPG才能顯示到更高的解析度
