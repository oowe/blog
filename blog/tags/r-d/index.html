中文文件編號格式#

  1. 第一層編號為國字大寫(壹、貳、參、肆、伍)，不縮排
  2. 第二層編號為國字小寫(一、二、三、四、伍)，縮排兩字元，內文首行縮排兩字元
  3. 第三層編號為小括號及國字小寫組成，後面不得加頓號[(一))(二)(三)(四)(五)]，不縮排
  4. 第四層編號為阿拉伯數字及小數點組成(1. 2. 3. 4. 5.)，縮排兩字元，內文切齊不縮排

Reference from https://kknews.cc/zh-tw/n/y35z9vn.html

info

「一」「二」「三」和「甲」「乙」「丙」等，以及阿拉伯數字，用括號括起來，表示序次。帶括號的序次後面不能再用頓號或逗號。

英文文件編號格式#

Reference from https://www.editing.tw/IEEE%E8%8B%B1%E6%96%87%E8%AB%96%E6%96%87%E6%A8%99%E9%A1%8C-Title-%E6%91%98%E8%A6%81-Abstract-%E5%8F%83%E8%80%83%E6%96%87%E7%8D%BB-Reference%E6%A0%BC%E5%BC%8F

  1. 第一標題：以羅馬數字編號，置於段落的中間上方，所有字母大寫。
    =>羅馬數字(I, II, III, IV, V, VI, VII, VIII, IX, X)

  2. 第二標題：以大寫字母編號，遵循首字母大寫原則並斜體，靠左對齊。
    =>大寫字母(A,B,C,D,E)

  3. 第三標題：以阿拉伯數字編號，內縮行距為一個破折號的長度（Em Dash「—」，數字後加上冒號，倘若數字後面緊接標題，則在標題後加上冒號。標題文字的部分，則遵循首字母大寫原則。描述文字可直接在冒號後面，不需另起一行。
    =>阿拉伯數字(0,1,2,3,4,5)

  4. 第四標題：和第三標題格式相同，但是內縮行距為兩個破折號的長度（Em Dash「—」，標題文字遵循句首大寫原則。
