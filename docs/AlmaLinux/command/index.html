安裝 EPEL (Extra Packages for Enterprise Linux)#

info

Reference from https://coodie-h.blogspot.com/2017/09/centos7epel-repository.html
EPEL（Extra Packages for Enterprise Linux）是由 Fedora Special Interest Group 社群所維護的套件庫，這個套件庫可用於 Red Hat 系列的 OS，如 RHEL、CentOS，因官方的套件庫常找不到需要的套件，所以安裝 EPEL 是蠻需要的。

sudo yum install epel-releasesudo yum update

安裝套件指令#

sudo yum update -y

安裝 NodeJS#

sudo yum install -y nodejs

新增一般使用者#

sudo useradd node //會自動建立家目錄
