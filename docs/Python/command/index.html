def foo1(value):    if value:        return value    else:        return None
def foo2(value):    """纯return语句，相当于`return None`"""    if value:        return value    else:        return

Init .py#

  • import package 時就會執行的檔案
  • 因為 requests 目錄帶有init.py，所以可以將 requests 看成是一個 regular package，當 requests 被其他 module import 時，預設會執行init.py

tuple、list#

  • tuple
    • 是屬於不可變的資料型態(immutable)，當你的值都不會改變，而且你也不希望值被更動的時候，就是適合使用 tuple 的時候
    • 以圓括號代表，例如 a = (1,2,3)
    • 要特別注意若只有一個 tuple 元素要加,才會成立 (11,)，若(11)代表整數 11
    • 占用的空間比較少
  • list
    • 是可變，可使用 append()、remove()、pop()等會更動值的操作。
    • list 是 b = [1,2,3]

類別、建構式、方法、物件#

  • 類別命名通常使用 Pascal 命名法，首字大寫，不使用空白和底線。
#汽車類別class Cars:    #建構式    def __init__(self, color):        self.color = color    #方法    def drive(self):        print(f"My car is {self.color} color")
#建立物件a = Cars()

pytest assert 和 raises 的用法#

-python3 -m pytest -v test -v: 顯示結果[PASS] OR [FAILED] -s: LOG 及細節全部顯示 都不加: 只有顯示完成%數

  • raises： 在斷言一些代碼塊或者函數時會引發意料之中的異常或者其他失敗的異常，導致程序無法運行時，使用 raises 捕獲匹配到的異常，可以繼續讓代碼正常運行。源碼：

    • 放在 python_env/lib/python3.8/site-packages/_pytest/python_api.py

  • with pytest.raises(ValueError) as excinfo:

    • 當判斷有 ValueError 時，代表測試正確。

  • valueError 可以 print 出來

    with pytest.raises(ValueError) as excinfo:        client.add_report(callback=print,                          resource_id='myresource',                          measurement='voltage',                          report_name='non_existant')                          #report_name='TELEMETRY_STATUS')    print(f"aaaaaaaa{excinfo.value}")

startswith#

  • Python startswith() 方法用于检查字符串是否是以指定子字符串开头，如果是则返回 True，否则返回 False

async await 說明#

  • async / await 是 Python 3.5+ 之後出現的語法，讓 Coroutine 與 Coroutine 之間的調度更加清楚
  • async：用來宣告 function 能夠有異步的功能
  • await：用來標記 Coroutine 切換暫停和繼續的點
