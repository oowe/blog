EPRI VTN Install Step

EPRI VTN 安裝環境#

OS: Ubuntu20.04LTSJAVA: 7PostgreSQL: 11RVM: 1.29.12Bundler:1.12.5Ruby:Rail:

EPRI VTN 安裝步驟#

Download EPRI VTN github#

  1. cd Download/
  2. git clone https://github.com/epri-dev/OpenADR-Virtual-Top-Node.git
  3. mv OpenADR-Virtual-Top-Node ~/oadr

Install Java7#

  1. wget https://files-cdn.liferay.com/mirrors/download.oracle.com/otn-pub/java/jdk/7u80-b15/jdk-7u80-linux-x64.tar.gz
  2. tar zxf jdk-7u80-linux-x64.tar.gz
  3. sudo mkdir /usr/lib/jvm
  4. sudo mv jdk1.7.0_80/ /usr/lib/jvm
  5. sudo mv /usr/lib/jvm/jdk1.7.0_80/ /usr/lib/jvm/java-7-sun
  6. sudo apt install -y vim
  7. vim ~/.bashrc
  8. add text
    export JAVA_HOME=/usr/lib/jvm/java-7-sunexport JRE_HOME=${JAVA_HOME}/jreexport CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/libexport PATH=${JAVA_HOME}/bin:$PATH
  9. source ~/.bashrc
  10. sudo update-alternatives --install /usr/bin/javac javac /usr/lib/jvm/java-7-sun/bin/javac 300
  11. sudo update-alternatives --install /usr/bin/jar jar /usr/lib/jvm/java-7-sun/bin/jar 300
  12. sudo update-alternatives --config java
  13. java -version

Install PostgreSQL-11 版#

  1. wget --quiet -O - https://www.postgresql.org/media/keys/ACCC4CF8.asc | sudo apt-key add -
  2. RELEASE=$(lsb_release -cs)
  3. echo "deb http://apt.postgresql.org/pub/repos/apt/ ${RELEASE}"-pgdg main | sudo tee /etc/apt/sources.list.d/pgdg.list
  4. cat /etc/apt/sources.list.d/pgdg.list
    確認出現下列文字即可deb http://apt.postgresql.org/pub/repos/apt/ bionic-pgdg main
  5. sudo apt update
  6. sudo apt install -y postgresql-11

Setup username and create db in PostgreSQL#

  1. sudo -i -u postgres
  2. psql
  3. CREATE USER "{username}" WITH PASSWORD '{password}';
  4. CREATE DATABASE {dbname} ENCODING 'UTF8' LC_COLLATE='C' LC_CTYPE='C' template=template0 OWNER "{username}";
  5. exit
  6. exit

Install pgAdmin4#

info

若發生憑證無效，請檢查是否有安裝 ca-certificates

  1. sudo apt install -y curl ca-certificates
  2. sudo curl https://www.pgadmin.org/static/packages_pgadmin_org.pub | sudo apt-key add
  3. sudo sh -c 'echo "deb https://ftp.postgresql.org/pub/pgadmin/pgadmin4/apt/$(lsb_release -cs) pgadmin4 main" > /etc/apt/sources.list.d/pgadmin4.list && apt update'
  4. sudo apt install -y pgadmin4
  5. sudo /usr/pgadmin4/bin/setup-web.sh
  6. enter email
  7. ener password
  8. y
  9. y
  10. y
  11. open browser http://192.168.11.237/pgadmin4

#General

#Connection
Host name/"localhost"
Maintenance database/{dbName}
Username/{username}
Password/{password}
save password

Install rvm#

info

若發生憑證無效，請檢查是否有安裝 ca-certificates

  1. sudo apt install -y gnupg2 curl ca-certificates
  2. gpg2 --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3 7D2BAF1CF37B13E2069D6956105BD0E739499BDB
  3. curl -sSL https://get.rvm.io | bash -s stable
  4. echo source ~/.rvm/scripts/rvm >> ~/.bashrc
  5. source ~/.bashrc
  6. rvm version
  7. echo 'gem: --no-document' >> ~/.gemrc

Install jruby1.17.10#

  1. wget https://rubygems.org/downloads/jruby-openssl-0.9.4.gem
  2. sudo apt install -y rubygems
  3. sudo gem install jruby-openssl
  4. cd ~/oadr
  5. rvm install "jruby-1.7.10"
  6. rvm alias create default jruby-1.7.10
  7. rvm use
check output is
Using /home/user/.rvm/gems/jruby-1.7.10 with gemset oadr

Install bundler#

  1. gem sources --remove https://rubygems.org/
  2. gem sources --add http://rubygems.org/
  3. y
  4. gem sources -l
check output is
*** CURRENT SOURCES ***
http://rubygems.org/
  1. gem install bundler:1.12.5
  2. modify Gemfile
changesource 'https://rubygems.org'tosource 'http://rubygems.org'
  1. wget https://raw.githubusercontent.com/rubygems/rubygems/master/lib/rubygems/ssl_certs/rubygems.org/GlobalSignRootCA_R3.pem
  2. mv GlobalSignRootCA_R3.pem ~/.rvm/rubies/jruby-1.7.10/lib/ruby/shared/rubygems/ssl_certs/
  3. gem update --system
  4. bundle _1.12.5_ install

安裝 torquebox#

  1. sudo mkdir /opt/torquebox
  2. sudo chown {username}:{username} /opt/torquebox
  3. Download torquebox-dist-3.0.2-bin.tar.gz

https://drive.google.com/uc?export=download&id=1FFilC4-gN84ELvWzYdj6IikVXjK1pYIa

  1. tar zxf torquebox-dist-3.0.2-bin.tar.gz
  2. mv torquebox-3.0.2 /opt/torquebox
  3. cd /opt/torquebox
  4. ln -s torquebox-3.0.2 current
  5. vim ~/.bashrc
#addexport TORQUEBOX_HOME=/opt/torquebox/currentexport JBOSS_HOME=$TORQUEBOX_HOME/jbossexport JRUBY_HOME=$TORQUEBOX_HOME/jrubyPATH=$JBOSS_HOME/bin:$JRUBY_HOME/bin:$PATHexport RAILS_ENV=production
  1. source ~/.bashrc
