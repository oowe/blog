Download visual studio code#

https://code.visualstudio.com/Download

Install visual studio code#

sudo dpkg -i {filename}

好用套件#

  1. Material Theme
  2. Prettier /setting/txt editor/Formatting/Format on save
  3. Bracket pair colorizer 2
  4. auto rename tag
  5. css peak
  6. power mode /enable function and disable shake and decrease time of function
  7. Bookmarks ctrl+alt+k : 建立/刪除書籤 ctrl+alt+j : 到上一個書籤 ctrl+alt+l : 到下一個書籤
  8. Path Intellisense
  9. vscode-icons-mac 更換主題的方式有兩種 一種是 按 F1 或是 Ctrl + Shift + P 後 輸入 icon 選擇 喜好設定:檔案圖示佈景主題 就會出現你有安裝過的所有檔案圖示佈景主題再做選擇即可。

中文化#

ctrl+p 輸入 ext install MS-CEINTL.vscode-language-pack-zh-hant 套用重開即可

快捷鍵#

  1. alt+上下 (將指令上下移動)
  2. ctrl+alt+上下 (將 ubuntu 桌面切換)
  3. 空選定狀態下按 ctrl+c 就是複制行
  4. 空選定狀態下按 ctrl+x 剪切命令來删除一行
  5. ctrl+(開新的編輯視窗)
  6. ctrl+j(開啟關閉終端機視窗)
  7. ctrl+home(跳至文件第 1 行)
  8. ctrl+end(跳至文件最後)
  9. 若要多新增一個資料夾和文件，可以直接建立檔案(new_folder/new_file)
  10. F12 快速移到 function 定義
  11. Ctrl+ Alt + 減號 （返回上一個位置）

編輯 c_cpp_properties.json先按Ctrl+Shift+P再輸入C / Cpp：Edit Configurations
如果要建立自己的應用程式，請建立task.json先按Ctrl+Shift+P再輸入Tasks: Configure TasksCreate tasks.json file from templates選擇 other
{  // See https://go.microsoft.com/fwlink/?LinkId=733558  // for the documentation about the tasks.json format  "version": "2.0.0",  "tasks": [  {  "label": "echo",               //可以敘述這個程是在幹麻的，非必填  "type": "shell",  "command": "echo Hello"          //若是C++ compiler 請填"gcc", 若是python 請填''python"  }  ]}如果你想讓vs code可以ctrl+shift+b 可以compile C++ code請加入到build group像下面這樣
{  "version": "2.0.0",  "tasks": [  {  "label": "build hello world",  "type": "shell",  "command": "g++",  "args": [  "-g", "-o", "${fileBasenameNoExtension}.out", "${file}"  ],  "group": {  "kind": "build",  "isDefault": true  }  }  ]}
